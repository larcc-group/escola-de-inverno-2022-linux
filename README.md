# Bem vindo à aula de introdução ao linux da Escola de Inverno de 2022

### Para que você possa acompanhar a aula e realizar as atividades práticas, é fortemente recomendado baixar e instalar alguns itens antecipadamente.

Baixe o VMware para podermos criar as VMs (Máquina Virtual). Acesse [aqui](https://customerconnect.vmware.com/en/downloads/details?downloadGroup=WKST-PLAYER-1623-NEW&productId=1039&rPId=85399).\
Faça o download da ISO do sistema operacional Ubuntu que iremos criar a VM. Acesse [aqui](https://ubuntu.com/download/desktop/thank-you?version=22.04&architecture=amd64).\
Faça o download do Putty para podermos acessar a VM via ssh. Acesse [aqui](https://the.earth.li/~sgtatham/putty/latest/w64/putty.exe).
